Socials

socials - это БЭМ блок, который используется для создания списков социальных сетей. Что бы сделать ссылки доступными мы НЕ ИСПОЛЬЗУЕМ span.visually-hidden с вложенным в него текстовым описанием, мы используем aria-label.

Pagination

pagination - это БЭМ блок, который используется для создания пагинации. Список ссылок на страницы и контролы разделены, это не лжин список! В списоке ссылок, выбранный элемент сделан тегом span, а не ссылкой, на элементе висит модификатор pagination__link--current. Ссылкам обязательно прописывать aria-label так как номера страниц не информативны.

Breadcrumbs

breadcrumbs - это БЭМ блок, который используется для создания "хлебных крошек". .breadcrumbs__list - это список, в последнем элементе списка, если это текущая страница, вставлен span, а не ссылка, на элементе висит модификатор breadcrumbs__link--current.

Lists

list - это БЭМ блок, который используется для создания списков. Для создания нумированных списков используем ol, ненумерованных ul это важно несмотря на то что мы обнуляем стили списков. В простых списках где нет сложной вложенности, текст вкладывается прямо в li, на li не вешается класс, всё стилизуется от родителя.

list--numeric - это модификатор для нумерованных списков. Для передачи кастомных значений в нумерацию можно использовать data атрибуты. Например на li повесить data-num="кастомное значение", а для псевдоэлемента в css прописать content: attr(data-num);
