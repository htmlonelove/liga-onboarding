Шапка проекта

  1. Элементы списка контактов не переносятся
  2. При разрешении переноса в списке контактов отсутствуют отступы между рядами
  3. При удалении списка контактов меню прилипнет к логотипу
  4. У списка контактов отсутствуют отступы, при переполнении или уменьшении ширины экрана список контактов прилипнет к лого и меню
  5. В выпадающем меню не ограничена высота, список может быть больше высоты экрана

Хлебные крошки

  1. У элементов списка отсутствуют отступы вниз, при выстраивании сетки в две строки элементы слипаются
  2. Стрелка у последнего элемента убирается через модификатор, если на поледний элемент не повестить модификатор (по причине того что он должен быть активным), то у него сохранится стрелка
  3. У списка установлен justify-content: space-between; Это свойство крайне не рекомендуется устанавливать в блоки с возможным переполнением, так как при переполнении элементы должны идти с лева на право если иное поведение не предусмотрено дизайном.

Текстовый блок

  1. У последнего текстового блока text-block-broken не удалён отступ снизу
  2. В первом текстовом блоке у параграфа отсутствует отступ, при добавлении новых параграфов они слипнуться
  3. Во втором текстовом блоке, позиционирование второго параграфа сделано с помощью nth-child(n), при добавлении новых параграфов они не будут идти в шахматном порядке
  4. Во втором текстовом блоке, отступы второго параграфа сделаны с помощью nth-child(n), при добавлении новых параграфов они будут слипаться

Карточки товаров

  1. У списка установлен justify-content: space-between; Это свойство крайне не рекомендуется устанавливать в блоки с возможным переполнением, так как при переполнении элементы должны идти с лева на право если иное поведение не предусмотрено дизайном
  2. При переполнении списка отсутствует отступ между рядами карточек, карточки слипаются
  3. У карточки не указан overflow: hidden; картинка вывалиывается из карточки
  4. При вставке высокой картинки карточка растягивается
  5. У контентного блока в карточке отсутствует внутренний отступ сверху, при удалении заголовка описание прилипнет к картинке
  6. При переполнении контентного блока карточки не подстраиваются под высоту максимальной карточки
  7. Если поправить высоту карточек при переполнении то низ растянутых карточек будет пустой
  8. Между термином и определением отсутствует отступ

Пагинация

  1. У ссылок отсутствуют отступы
  2. У ссылок зафиксирована ширина
  3. У списка есть отступ с левой стороны. При удалении ссылки назад список не будет прижат к краю.
  4. При переполнении и переносе на новую строку ссылки слипаются

Подвал проекта

  1. Между блоками в футере отсутствуют отступы, при переполнении они буду слипаться
  2. При переполнении соц сетей, смещается центральный блок с ссылками
